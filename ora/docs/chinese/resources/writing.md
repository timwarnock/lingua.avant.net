Learn [phonetics and pronunciation](pronunciation.md) for the Rosary prayers in traditional characters.

For actual writing and typing Chinese, pinyin and zhuyin serve as practical input tools.

## Input Methods and Typing

If you can type well on a Latin keyboard, you will be able to type Chinese well using pinyin. Pinyin is faster for typing, and it is the official system in mainland China. You will see pinyin on signs throughout China, which can be useful for travel and navigation.

Zhuyin remains popular in Taiwan, where it is a default input method on smartphones and computers.

In practice, input methods often let you omit tone marks (the system guesses from context). You can also use numerical tones such as zhong1 wen2 for quick typing. Diacritic marks like zhōngwén are common; for ambiguous cases use an apostrophe, e.g. chán'gān versus cháng'ān.

Learning to write and type the prayers and study notes benefits from practice with either system. Many learners use apps and repetition to build automaticity with input.
